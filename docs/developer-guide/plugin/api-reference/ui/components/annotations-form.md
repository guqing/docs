## Props

| 属性名  | 类型                              | 默认值   | 描述                                           |
|---------|-----------------------------------|----------|------------------------------------------------|
| `group` | string                          | *无默认值* | 定义组件所属的分组。                           |
| `kind`  | string                          | *无默认值* | 定义组件的种类。                               |
| `value` | { [key: string]: string; } \| null | null   | 可选。包含键值对的对象或空值，用于存储数据。   |
